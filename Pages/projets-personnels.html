Bot Discord “Moksi's Bazaar”

Moksi's Bazaar est un bot Discord entièrement développé en JavaScript (discord.js), qui offre à la fois un système de monnaie virtuelle et plusieurs mini‐jeux de casino : blackjack, roulette et machine à sous. Les utilisateurs dans un serveur Discord peuvent miser, gagner des crédits et suivre leur classement en temps réel.

  • Compétences : API Discord, programmation événementielle, gestion des permissions
  • Techniques : Python (discord.py), hébergement, gestion d’events, logs, SQL

Fonctionnement du Bot

Le bot est hébergé sur Railway grâce à une instance GitHub 24/7. Une base de données PostgreSQL (intégrée à Railway) stocke en temps réel :

  • Les identifiants Discord de chaque utilisateur
  • Les soldes (crédits) et leur historique
  • Les statuts de participation aux différents jeux (mise, gains/pertes)

Dès qu’un utilisateur envoie une commande (par exemple !blackjack ou !roulette), le bot interroge la table users pour vérifier le solde, met à jour les données appropriées selon les résultats, puis renvoie un message Discord détaillé (embed) affichant l’évolution du profil et du classement.
