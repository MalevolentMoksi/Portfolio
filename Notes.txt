Mobile UI :

Ensure Fluid Layouts

    Make all images and content blocks scale within their containers:

    img, section img, .presentation-block, .project {
      max-width: 100%;
      height: auto;
    }

    Use relative units (%, rem, em) instead of fixed pixels for widths, margins and paddings.

Add a Mobile Breakpoint
Append to your style.css (at the very end) a media query for screens narrower than, say, 768 px:

/* Mobile styles */
@media (max-width: 768px) {
  /* Simplify/hide heavy backgrounds for performance */
  #background,
  #particles-js {
    display: none;
  }

  /* Stack nav links vertically */
  nav ul {
    flex-direction: column;
    gap: 0.75rem;
    padding: 1rem;
  }

  /* Increase tap targets */
  nav ul li a {
    display: block;
    padding: 0.75rem;
    font-size: 1.25rem;
  }

  /* Make header text more compact */
  header h1 {
    font-size: 1.75rem;
    margin-bottom: 0.5rem;
  }
  header h3, header h4 {
    font-size: 1rem;
  }

  /* Single-column grid for your “Présentation” */
  .presentation-grid {
    grid-template-columns: 1fr;
  }

  /* Reduce margins/paddings to fit small screens */
  main {
    margin: 1rem;
    padding: 1rem;
  }
  main section {
    padding: 1rem;
  }

  /* Project list: images full-width, text below */
  #project-list article {
    display: flex;
    flex-direction: column;
    align-items: stretch;
  }
  #project-list article img {
    width: 100%;
    margin-bottom: 0.75rem;
  }

  /* Back-to-top button: smaller and closer to edge */
  #back-to-top {
    bottom: 10px;
    right: 10px;
    padding: 8px 12px;
    font-size: 1.25rem;
  }
}

Optional: Hamburger Menu for Nav
If you’d like to collapse your navigation into a “hamburger” on mobile, here’s a lightweight pattern:

HTML (inside <header>):

<input type="checkbox" id="nav-toggle" hidden>
<label for="nav-toggle" class="nav-toggle-label">
  <span></span><span></span><span></span>
</label>
<nav>
  <ul>…</ul>
</nav>
